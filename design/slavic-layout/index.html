Slavic Layout

Славянская раскладка

The Slavic Layout is a keyboard layout for Slavic languages. It allows you to input special characters of Serbian, Belarusian, Ukrainian and Macedonian languages in the familiar Russian keyboard layout ЙЦУКЕН.

The Slavic Layout is based on Typographic Layout and supports most of the characters used in it.

С этой раскладкой вы сможете вводить специальные символы сербского, белорусского, украинского и македонского языков в привычной русской раскладке ЙЦУКЕН.

Раскладка создана на основе Типографской раскладки Ильи Бирмана и поддерживает большинство использующихся в ней символов.

Выделенные символы набираются с помощью клавиши Alt.¹

Highlighted characters are typed using the Alt key.¹

Символы, использующиеся на этих клавишах в раскладке Бирмана, соответственно, не работают.

Characters used on these keys in Birman's layout do not work.
